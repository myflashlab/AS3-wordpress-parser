example_FlashDevelopSampleAndroidApp ReadMe:
-----------------------------------------------------

This example file has been coded inside the FlashDevelop IDE (http://www.flashdevelop.org/)
you can copy the sample APK file (example_FlashDevelopSampleAndroidApp\dist\wpparser.apk) into your
Android mobile device and see a simple usage of wordpress codes in an application!


NOTE: This APK project is only for the sake of an example, the main wordpress codes are just some abstarct 
codes to control the pure wordpress data such as your posts, comments, etc...

This example file uses the abstract codes to create a simple Android WP app, you can also use the codes
in the similar way to create somethig else related to managing Wordpress data in your AS3 projects.
